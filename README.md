 ## 写这个项目的初衷
 
在期末考的时候大部分专业课老师是没有划重点的，由于没有补考期末考试容错很低，然后我们的复习范围又很广。对于平时有社交的同学就可以找学长学姐了解考试的重点，而对于i人来说就是面对网上碎片化信息。
 
 所以做这个项目的初衷——**让信息平权，让知识共享**。每个人都能获得好的学习资源和应试技巧，而不是靠人脉和信息差。
 
 我希望通过这些资料的整理，能帮大家节省更多复习时间，把精力花在探索更多可能性上。
 
 **打破信息差，是我们的目标。**
 
 ## 项目特点
 
 本项目的内容不止是资料的堆砌，每门课程的README都包含以下维度：
 
 ### 混及格 & 拿高分的思路
 每门课都会说明：最低要求是什么？想拿高分要怎么做？哪些章节是送分题，哪些是拉分题？
 
 ### 对老师的个人评价
 不同老师的授课风格、给分习惯、重点倾向——这些都是选课和备考的重要参考。
 
 ### 对课程的个人想法
 这门课到底有没有用？值不值得认真学？课程内容在工作和考研中能派上什么用场？
 
 ### 就业 / 考研 / 未来规划的思路
 结合本专业的实际情况，聊聊每门课对未来发展的影响，帮你早点想清楚方向。
 
 每个README里都有一个 **📌 重点看这里** 表格，那是每门课最核心的应试策略和避坑指南，建议优先阅读。
 
 另外，我也会把一些"邪修方法"（一些应试技巧和混平时分）陆续加进来。

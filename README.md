# 厦门大学嘉庚学院电子系 - 课程学习资源分享

> 电子类专业课程资料共享计划

本仓库整理了大学期间多门核心课程的学习资料，包括教材、实验报告、复习笔记、考试重点、课程代码等。

## 课程目录

| 课程 | 内容 |
|------|------|
| [人工智能](./人工智能) | 金融风险预测研究（论文+代码+数据集） |
| [信号与系统](./信号与系统) | 教材PDF、知识点总结、z变换笔记 |
| [单片机原理及应用](./单片机原理及应用) | STM32实验报告、课件PPT、考试重点 |
| [概论统计B](./概论统计B) | 概率论期末复习资料 |
| [电子线路非线性](./电子线路非线性) | 期中/期末复习PPT |
| [电路分析实验](./电路分析实验) | 11个实验报告+理论考试资料 |
| [电路设计与仿真](./电路设计与仿真) | LTspice仿真、期末复习资料 |

## 项目特点

- 每门课都有 README，包含选课建议和避坑指南
- 按课程分类整理，结构清晰
- 包含实验报告、复习资料、考试重点
- 持续更新中

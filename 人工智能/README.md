# 人工智能：金融风险预测研究

> 基于逻辑回归与朴素贝叶斯的贷款违约二分类预测 —— Python + scikit-learn

## 项目简介

利用机器学习方法对金融风险数据集进行二分类预测（是否违约），完成从数据清洗到模型评估的全流程。
这是本仓库中最能体现代码能力的技术项目，适合作为技术面试的讨论素材。

## 技术栈

| 类别 | 使用 |
|------|------|
| 语言 | Python 3 |
| 数据处理 | pandas, numpy |
| 机器学习 | scikit-learn (LogisticRegression, GaussianNB) |
| 可视化 | matplotlib, seaborn |
| 模型评估 | Accuracy, Precision, Recall, F1, ROC-AUC |

## 工作流程

1. 数据加载与探索（检查缺失值、数据分布）
2. 特征工程（缺失值处理、标准化 StandardScaler）
3. 数据集划分（70% 训练 / 30% 测试，分层采样）
4. 模型训练：逻辑回归 + 朴素贝叶斯
5. 模型评估：混淆矩阵、ROC曲线、多指标对比
6. 特征重要性分析（Permutation Importance）
